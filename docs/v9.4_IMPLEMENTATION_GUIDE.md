## BDI Cycle Integration

### Main Task Flow (Shogun.process_task)

```python
async def process_task(self, task: Task) -> Dict[str, Any]:
    """
    BDI統合フロー:
    1. 知覚 (Perceive): タスクと環境から信念を更新
    2. 熟慮 (Deliberate): 追求すべき願望を選択
    3. 計画 (Plan): ルーティング決定
    4. 実行 (Execute): 適切なエージェントで実行
    5. 再考 (Reconsider): 結果に基づき信念を更新
    """
    # BDI Step 1: Perceive
    if self.bdi_enabled:
        await self._bdi_perceive(task)

    # BDI Step 2: Deliberate
    selected_desire = await self._bdi_deliberate(task)

    # BDI Step 3: Plan
    intention = await self._bdi_plan(task, selected_desire)

    # BDI Step 4: Execute
    result = await self._execute_based_on_complexity(task)

    # BDI Step 5: Reconsider
    await self._bdi_reconsider(task, intention, result)

    return result
```

## Usage

### Basic Usage

```python
from core.system_orchestrator import SystemOrchestrator, SystemConfig, SystemMode

# Create config
config = SystemConfig(
    mode=SystemMode.BATTALION,
    claude_api_key="...",
    gemini_api_key="...",
    groq_api_key="...",
    tavily_api_key="..."
)

# Initialize
orchestrator = SystemOrchestrator(config)
await orchestrator.initialize()

# Process task (delegates to Shogun with BDI)
result = await orchestrator.process_task("Implement a Python inventory system")

# Get statistics
stats = orchestrator.get_all_statistics()
bdi_states = orchestrator.get_bdi_states()
```

### CLI Usage

```bash
# Start system
python main.py

# Run task with automatic routing + BDI
python cli.py --task "Python在庫管理システム実装"

# Specify complexity
python cli.py --task "アーキテクチャ設計" --complexity strategic
```
