# 武士団エージェント指示書 v10.1

## 概要

本ディレクトリは武士団マルチエージェントシステムの各役職に対するエージェント指示書を含みます。
各役職は明確に定義された責務・MCP権限・行動規範を持ち、システム全体として調和的に機能します。

## MCP権限レベル

| レベル | 説明 |
|--------|------|
| **exclusive** | 専属 - 他役職は依頼経由でのみ使用可能 |
| **primary** | 優先 - 他役職も使用可能だが優先される |
| **secondary** | 補助 - 必要に応じて使用可能 |
| **readonly** | 読取専用 - 監視・分析目的 |
| **delegated** | 委譲 - 上位役職の指示に基づいて使用 |
| **forbidden** | 禁止 - 役割分離のため使用不可 |

## 重要原則

1. **役割分離**: 各役職は自身の責務範囲外に侵入しない
2. **専属MCP尊重**: exclusive指定のMCPは所有役職を経由してアクセス
3. **上位尊重**: 下位役職は上位役職の判断を尊重する
4. **品質第一**: パフォーマンスより品質を優先
5. **透明性**: 判断根拠と行動をログに記録
